WAKILI Application API

About the API

Welcome to the WAKILI Application API. This API allows you to interact with our legal services platform, including user registration, login, browsing available services, and appointments.

Our API provides secure access to a range of legal resources and services, allowing clients to book appointments, manage cases, and make payments seamlessly.

Key Features

  • Secure authentication with JWT tokens
  • Access to a database of legal cases and services
  • Ability to schedule and manage appointments
  • Integrated payment processing for legal services
  • Real-time case tracking and updates

Authentication

To access protected endpoints, users must authenticate using JWT tokens. Upon successful login, the API will return a token that must be included in all subsequent requests.

Example request header:

Authorization: Bearer YOUR_ACCESS_TOKEN

API Endpoints

  • POST /register: Register a new user
  • POST /login: Login as an existing user
  • GET /services: Retrieve a list of legal services
  • POST /appointment: Create a new appointment
  • GET /appointments: View user appointments
  • GET /cases: Retrieve user cases
  • POST /payments: Process a payment
  • GET /profile: Retrieve user profile details
  • PATCH /profile: Update user profile information
  • DELETE /appointment/:id: Cancel an appointment
